## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-get-available-sensors`
- `allow-start-sensors`
- `allow-stop-sensors`
- `allow-registerListener`
- `allow-unregister-listener`
- `allow-get-grouped-sensor-data`
- `allow-create-group`
- `allow-get-group`
- `allow-get-groups`
- `allow-delete-group`
